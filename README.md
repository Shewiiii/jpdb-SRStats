# Jpdb SRStats
**Not functional yet**  
Get useful data from a [jpdb](https://jpdb.io/) JSON file, using [FSRS](https://github.com/open-spaced-repetition/py-fsrs) and its Optimizer (based on the entire review history) to get reliable data.
